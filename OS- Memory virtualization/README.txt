To run this program just type make in you unix terminal. ( this will compile the code and generate an executable calles memsim) 
then run the following:
 ./memsim <tracefile> <nframes> <lru|fifo|vms> <p> <d|q> 

tracefile: name of file
nframes: number of physical memory frames
<lru|fifo|vms>: algorithm 
<p> : percentage
<d|q>: make sure this is specified otherwise the code will terminate.
